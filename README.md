# OData Deep Dive Server

This repo contains the CAP Node.js based project that is home to the server
providing the OData service for the OData Deep Dive series of tutorials. For
more info, see the blog post [OData Deep Dive rewrite in the
open](https://qmacro.org/blog/posts/2026/02/02/odata-deep-dive-rewrite-in-the-open/).

## Support

Support for the content in this repository is available via issues that can be
raised on the corresponding tutorials in the [OData Deep
Dive](https://developers.sap.com/mission.odata-deep-dive.html) mission on the
SAP Tutorial Navigator.
